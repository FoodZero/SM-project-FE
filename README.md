# SM Project FrontEnd

이 프로젝트는 **React Native** 기반의 프론트엔드 애플리케이션입니다.

## 주요 기능
1. **회원 가입 및 로그인**
2. **냉장고 관리 기능**
3. **식재료 관리 기능**
4. **OCR을 활용한 영수증 스캔으로 냉장고 식재료 등록 기능**
5. **냉장고 공유 기능**
6. **레시피 검색 기능**
7. **AI 레시피 추천 기능**
8. **레시피 북마크 및 추천 기능**
9. **커뮤니티 게시글 관리**
10. **커뮤니티 게시글 댓글 기능**
11. **위치 등록 기능**
12. **앱 푸시 알림 기능**
